Project Overview

The Nienkamper Furniture App Design project aimed to translate the existing Nienkamper website experience into a mobile application, enhancing user accessibility and experience. The goal was to maintain brand consistency while optimizing the interface for mobile use, providing users with a seamless transition between the website and the app. The app design is made using Figma, Photoshop, Coursel and Aero. This project also explores the Augmented reality by using 3D models of chair.

Creative Process

I started with Low-fidelity wireframes to High-fidelity wireframes. For the app wireframes, I tried to keep it as similar as the original website. I made few changes to make the user experience better like burger menu, filter options etc. Once the wireframes were done, the next step was to make a prototype, as it give real look and feel of an app.

Next step of this project was to try new technology, that is , Augmentedreality. For the AR/VR experience, I use online model-viewer, aero and 3D substance app. As it was my first time using it, the result was quite different from my thoughts. But it was fun laerning about new technology.
