Project Overview

The Minimalist Jewellery Website is an online platform showcasing a hypothetical jewellery brand called "Minimalist." The website aims to provide a visually appealing and user-friendly experience for visitors interested in exploring minimalist-inspired jewellery pieces. It features a clean and modern design, incorporating elements of simplicity and elegance to reflect the brand's aesthetic.

This project was a valuable learning experience that allowed me to show my front-end development skills while showcasing my creativity in design.

Coding

I have coded this website in Visual Studio, using HTML, CSS and Javascript. Given that the brand name was "Minimalist," I aimed to reflect this ethos throughout the website's visual presentation. The color scheme was kept simple and clean, with predominantly white backgrounds and accents of soft, muted tones to complement the jewellery pieces. Their are 3 pages are that working. I have used free server to host my website as I am still a beginner.
